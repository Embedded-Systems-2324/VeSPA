# Vivado Folder
This folder contains the Vivado project files.

## ps2_repo
This repository contains the PS/2 AXI4 Lite IP files. To use it, verify the user repositories in your IP Catalog and make sure "ps2_repo" is included.

## PS2 AXI
This folder contains the Vivado PS/2 files. The PS/2 AXI4 Lite is already included in the project, using the module in the "top_ps2.v" file, the program can be used in a more abstract way to the user.
