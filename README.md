# UART AXI4 Lite

## Project Overview
The UART AXI4 Lite project is dedicated to implementing a UART using the AXI4 Lite bus architecture. Specifically designed to function as a peripheral within a System-on-Chip, this project contributes to the development of the SoC for the VeSPA processor, offering a flexible and efficient solution for facilitating communication within embedded systems."

Feel free to explore the documentation and project files to gain a deeper understanding of the project. Contributions and feedback are always welcome!
