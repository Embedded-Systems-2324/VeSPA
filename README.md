# PS/2 AXI4 Lite

## Project 
The PS/2 AXI4 Lite project consists in implementing a module capable of detecting and identifying pressed keys in a PS/2 keyboard using the AXI4 Lite bus architecture. Designed to be used as a peripheral within a SoC, it provides a simple and efficient method to connect a keyboard to the Soc for the VeSPA processor.

Feel free to explore the documentation and project files to gain a deeper understanding of the project. Contributions and feedback are always welcome!
